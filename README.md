Dataset Details:
A survey includes oquestions about video game habits and questions from Buss-Perry Agreesion Questionnarie(BPAQ)

There are 43 variables in the dataset:
* Timestamp
* 5 Demographic questions
* 37 Behavioral and BPAQ questions

BPAQ: The 1992 version of the AQ is a 29-item questionnarie in which participants rank certain statements along a 5-point continuum

Participants: There are 147 participants with age 15 to 32. Both gender female and male are included.

Dataset is from gaggle: https://www.kaggle.com/datasets/muhammadshamoeel/effects-of-video-games-on-aggression-msdos-csv/data
